Cronograma

Desarrollo Fullstack

Desarrollado para el IPRED UIS

Actualmente hay una versión pública en: [IPRED UIS].

De esta vista hay 3 versiones: La pública, La vista decronograma que edita el horario del semestre actual y la vista de cronograma encargada de planificar el cronograma del siguiente semestre.

La vista mas completa es la que permite planificar las actividades, permite crear filas. Y el CRUD de las actividades, verificando en el proceso los cruces horarios. Ademas de permite filtrar el contenido visualizado y otras herramientas como modo pantalla completa, cuenta la cantidad de actividades programadas. En caso de que intentes crear o editar una actividad y tenga un cruce horario, sugiere dias en los que no tendrá cruces horarios.

Smart Trash

Desarrollo fullstack y Arduino

Desarrollado para microcontroladores UIS

Fotos y video adicionales: [Click aquí].

El proyecto es un separador de basura, especificamente de Metal, Vidrio, Plastico e indefinido.

El prototipo en arduino detectaba a traves de un sensor inductivo para reconocer si era de metal, y para diferenciar entre vidrio y plastico se atravesaba el material con un laser y al otro lado se recolectaba la información a traves de un fotosensor. Una vez se reconoce que material es, se aprovecha la gravedad para dejar caer el elemento en el compartimento respectivo. Una vez esto sucede los sensores ultrasonicos detectaban el nuevo nivel del compartimento y enviaba la informacion a la base de datos de Firebase y esta informacion era mostrada a traves de una pagina web.

Tecnologías:
  • Arduino
  • Bootstrap
  • jQuery
  • NodeJS
  • Firebase
