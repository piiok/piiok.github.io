Acerca de mi

Ruta profecional

Mi nombre es Paola Andrea Caicedo Gualdrón, actualmente me encuentro cursando 10mo semestre en la UIS y estoy desarrollando mi proyecto de grado, el cual consiste en una plataforma para la gestión del ciclo de vida de un producto biomédico concretamente implantes para pacientes específicos (PSI) en el área craneal. Para ello estamos realizando con mi compañero de proyecto de grado una SPA (Single Page Application) en Django junto con la librería API REST para la parte backend y para la para la parte de front ReactJS, con base de datos en MySQL.

Tambien soy auxiliar administrativa y desempeño el rol de desarrolladora fullstack en la plataforma administrativa para la sección de la UIS que se encarga de los pregrados a distancia (IPRED) en este cargo desarrollo en lenguajes tales como JQuery, ReactJS y Bootstrap 4 para la parte de frontend y Postgres, PHP con librerías tales como Smarty, y DomPDF para la parte de backend.

Por cuenta propia he realizado diversos proyectos, ya que soy una persona curiosa y autodidacta, me gusta aprender tecnologías nuevas.
